# Principal_Component_analysis
# 📈 Principal Component Analysis (PCA) Project

A beginner-friendly Machine Learning project demonstrating **Principal Component Analysis (PCA)** for dimensionality reduction and data visualization using Python.

## 🚀 Features

* Data Standardization
* PCA Implementation
* Dimensionality Reduction
* Variance Analysis
* Data Visualization
* Easy-to-understand Code

## 🛠️ Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Scikit-learn

## 📌 Project Objective

To reduce high-dimensional data into fewer components while preserving maximum information and improving visualization.


## 📊 Output

* Reduced Dimensions
* Explained Variance Ratio
* PCA Scatter Plot Visualization
